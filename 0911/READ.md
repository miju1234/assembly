1. **8비트 이진수의 MSB**  
   → 가장 왼쪽 비트 (비트 7)

2. **부호없는 이진수 → 10진수**  
   - 00110101 = 53  
   - 10010110 = 150  
   - 11001100 = 204

3. **이진수 덧셈**  
   - 10101111 + 11011011 = 1 1000 1010 (0x18A = 394)  
   - 10010111 + 11111111 = 1 1001 0110 (0x196 = 406)  
   - 01110101 + 10101100 = 1 0010 0001 (0x121 = 289)

4. **이진수 뺄셈**  
   - 00001101 − 00000111 = 00000110 (6)

5. **자료형 비트 수**  
   - word = 16비트  
   - doubleword = 32비트  
   - quadword = 64비트  
   - double quadword = 128비트

6. **주어진 10진수를 표현하는 최소 비트 수**

   - 42319 → 16비트
